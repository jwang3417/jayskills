# jayskills

Useful agent skills for myself.

## Skills

- [`jianw-cv-build`](./jianw-cv-build) — Set up the LaTeX toolchain (BasicTeX, the `res` resume class, Palatino fonts) and build `jianw_cv.pdf` from `jianw_cv.tex` on a fresh macOS machine.
- [`xiaohongshu-favorites-download`](./xiaohongshu-favorites-download) — Download all image-text posts from a user's Xiaohongshu / RedNote 收藏 (favorites) to local disk via QR-code login.

Each skill lives in its own directory with a `SKILL.md` and supporting `scripts/` (and `assets/` where needed).
